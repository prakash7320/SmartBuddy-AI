# SmartBuddy AI

SmartBuddy AI is an AI-powered chatbot built using FastAPI, HTML, CSS, and JavaScript.

## Features

* AI Chat using Groq API
* Image Analysis (Vision AI)
* Live Technology News
* Responsive User Interface
* FastAPI Backend
* Deployed on Render & Vercel

## Tech Stack

### Frontend

* HTML
* CSS
* JavaScript

### Backend

* FastAPI
* Python
* Requests

### APIs

* Groq API
* News API

## Installation

1. Clone the repository

git clone https://github.com/prakash7320/SmartBuddy-AI.git

2. Install dependencies

pip install -r requirements.txt

3. Create .env file

GROQ_API_KEY

NEWS_API_KEY

4. Run backend

uvicorn main:app --reload
